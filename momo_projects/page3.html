Techniques

Some of the patterns in these projects may be a bit more difficult, including the phoenix and dragon. These took more effort than many of the other projects, because of the detail involved. These projects will take longer, so you should plan ahead if you wish to make them.

Stitches and techniques used for these projects include the shell stitch which uses a series of double crochets into the same stitch to create a shell-like pattern. The popcorn stitch which involves making a cluster of stitches in the same stitch to create a raised texture. However, these projects also use many of the easier stitches as well, it's just a matter of how you use stitches that make a project look more complicated.
