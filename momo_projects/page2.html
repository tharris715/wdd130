Techniques

These projects use more basic stitches. These would be better choices for those newer to crochet. However they can still pose a challenge, especially with the smaller pieces, which need small and tight stitches, which can make it tricky to move the hook through the stitches without breaking the yarn. You should test the tensile strength of the yarn so you know if you can pull on it hard enough to make the small stitches.

Stitches and techniques used for these projects include the single crochet stitch, the most basic stitch in crochet. The half double crochet stitch which is slightly taller than the single crochet stitch. As well as the the double crochet stitch and treble crochet stitch which are both taller still.
